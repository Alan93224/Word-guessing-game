# 生成式 AI 字謎遊戲（Generative AI Word Guessing Game）

這是一個簡易的小遊戲，透過前端 React 介面連接 API，根據遊戲規則回答玩家的猜字謎問題。本專案包含前端 UI、API 呼叫流程與基本的字謎遊戲邏輯。

---

## 🚀 專案安裝與執行步驟

以下教你從零開始建立一個 React 專案，並將遊戲程式碼放入 `App.jsx`。

---
